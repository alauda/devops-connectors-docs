# Introduction

## What is JFrog Connector

JFrog Connector is a specialized connector component that enables secure and convenient integration with JFrog Artifactory, allowing users to interact with JFrog Artifactory repositories without handling credentials in clients like `mvn`, `npm`, or `pip`.

Once JFrog Connector Component is deployed, users can:

- Create JFrog connectors to integrate with JFrog Artifactory instances
- Perform Maven, NPM, and PyPI operations in CI/CD pipelines or Kubernetes workloads without directly handling credentials

## Application Scenarios

The JFrog Connector allows you to perform package operations securely by:

- Managing credentials centrally rather than hardcoding them in clients
- Automatically injecting authentication during package operations through the proxy

This approach is particularly useful for:

- `CI/CD pipelines` or `kubernetes jobs` requiring access to JFrog Artifactory repositories
- Teams sharing JFrog Artifactory access without sharing credentials
- Environments requiring centralized management of JFrog Artifactory credentials without distributing credentials to each client
- Multi-format repository access: Maven, NPM, and PyPI registries through a single connector
