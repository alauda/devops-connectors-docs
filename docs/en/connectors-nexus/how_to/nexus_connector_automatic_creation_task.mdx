# Nexus Connector Automatic Creation Tekton Task

> Run the install command on this page **once per cluster** (admin only) to install the Task and the tool-image ConfigMaps. Day-to-day users only create TaskRuns — see the [how-to guide](./using_nexus_connector_automatic_creation_task.mdx) for end-to-end examples.

Set the variables, then run the command below:

- `REGISTRY`: the registry host that stores the `devops/nexus-connector-automatic-creation` image.
- `TARGET_NS`: the namespace where you want to install the Task. The tool-image ConfigMaps are installed in `kube-public`.
